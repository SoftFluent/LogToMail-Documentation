# Settings

LogToMail configuration file is named "*__LogToMail.exe.config__*" and stored in the application folder, by default: *%ProgramFiles(x86)%\SoftFluent\LogToMail*. It is an XML file composed of two sections:

* Mail Settings
* LogToMail Settings
